一：本方法搭建博客的优势

  1. 不需要服务器和域名，而是选择部署在Github上，没有成本
  2. 不需要学习前端框架也可以有好看的页面，Hexo拥有数量不少的开源主题，比如CrazyWong、NexT等

二：本文需要的工具

  • Nodejs
  • Hexo
  • Git
  • Vscode（任意编辑器都可）

选择 Git 默认编辑器
